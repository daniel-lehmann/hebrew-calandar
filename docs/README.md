# Feature explanations

This directory explains how the Hebrew Calendar app works and what each feature does.

## Contents

- **[Calendar](calendar.md)** — Combined Gregorian/Hebrew month view, navigation, and view toggle
- **[Holidays](holidays.md)** — Holiday display and year-navigation arrows
- **[Parashot](parashot.md)** — Weekly Torah readings on Saturdays and the detail popup
- **[Search](search.md)** — Quick search (Ctrl+J) for holidays and dates
- **[Stats](stats.md)** — List Hebrew date by year, day-of-week histogram, and holiday line graph
- **[Mobile](mobile.md)** — Responsive layout and touch-friendly controls
